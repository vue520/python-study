Python入门教程

1. Python简介

Python是一种解释型、面向对象、动态数据类型的高级程序设计语言。

特点：

  • 简单易学
  • 免费开源
  • 跨平台
  • 丰富的库

2. 环境搭建

下载地址：python.org

验证安装：

python --version

3. 基础语法

第一个Python程序：

print("Hello, World!")

4. 数据类型

常用数据类型：

# 数字
a = 10
b = 3.14

# 字符串
s = "Python"

# 列表
list = [1, 2, 3]

# 字典
dict = {"name": "Alice", "age": 25}

5. 流程控制

if语句：

age = 18
if age >= 18:
    print("成年人")
else:
    print("未成年人")

for循环：

for i in range(5):
    print(i)

6. 函数

定义函数：

def add(a, b):
    return a + b

print(add(3, 5))  # 输出8
